# Pinokio-VideoPolish-er

1-click launcher for [VideoPolish-er](https://github.com/C0m3b4ck/VideoPolish-er) on [Pinokio](https://pinokio.computer/).

## What is VideoPolish-er?

VideoPolish-er is a CLI tool for transcribing and cleaning up audio/video files with automatic noise removal and stutter detection.

### Features

- 11 local AI transcription models
- Automatic silence removal
- Stutter detection and removal (uhm, uh, um, er, etc.)
- Verification against original script
- SRT, VTT, ASS, JSON output formats

## Installation

1. Install [Pinokio](https://pinokio.computer/)
2. Open Pinokio
3. Click **Install** and paste this URL:
   ```
   https://github.com/C0m3b4ck/Pinokio-VideoPolish-er
   ```

## Usage

After installation, click **Start** to launch VideoPolish-er Web UI.

## Custom Icon

Place your custom `icon.png` in this folder to replace the default icon.
